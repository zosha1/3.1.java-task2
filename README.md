**1. Формулу вычисления количества unit-тестов на основании значений, генерируемых JaCoCo**

v(G) = B - D + 1

где:
* V(G) - количество необходимых тестов;
* B - количество веток;
* D - количество точек принятия решений.

Для метода `nextStation` :

V(G) = 3 - 2 + 1

V(G) = 2

Для метода `prevStation` :

V(G) = 3 - 2 + 1

V(G) = 2
